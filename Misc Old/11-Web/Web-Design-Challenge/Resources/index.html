Summary: Latitude vs. X

In this assignment I was tasked with retrieving raw data from the Openweather API and break it out into useful informative data and graphs. After doing this, I was then tasked with filtering the data down to a list of potential vacation destinations I would likely want to visit due to ideal weather conditions I set for myself.

Per my research I determined that the closer a location is to the Earth’s equator, the higher it’s max temperature is. Outside of this finding, I was unable to produce strong correlations to wind speed, humidity levels, or cloudiness.
